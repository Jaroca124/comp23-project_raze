README

This is the README for Game 1 of Comp23 - Game Development.

Game Title: Project Raze
Creators: Jacob Caccamo, Samantha Welch, Brian O'Keefe
Date: 10/19/2015

Game Summary:
	A researcher located deep in the jungle was trying to outfit injured gorillas with advanced prosthetics. The gorillas, no-longer biological creatures, revolted, forcing the doctor to fend for his life.

Gameplay:
	This game combines the shooter and survival genres. The goal of the game is to survive as long as possible. Players can pick up health powerups and gun upgrades to help them survive.

Requirements:
	- Global Score Board
	- Realistic 2D Physics
	- JSON Tunable
	- Chiptune Music
